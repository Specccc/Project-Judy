# Project Judy

Project Judy is an AI-powered Discord bot built to provide modern server management with a personality.

Unlike traditional Discord bots, Judy combines moderation, automation, leveling, configurable member experiences, AI-powered conversations, and ambient interactions into a single, cohesive experience.

---

## Features

### Moderation
- Kick
- Ban
- Timeout
- Basic moderation tools
- Moderation logging

### Member Experience
- Custom welcome messages
- Custom goodbye messages
- Welcome DMs
- Auto Roles
- Configurable embeds
- Placeholder support

### Configuration
- Welcome channel
- Log channel
- Announcement channel
- SQLite configuration storage
- Live configuration overview

### Leveling
- XP system
- Automatic level progression
- Persistent SQLite storage

### AI
- Google Gemini integration
- Slash command AI chat
- Character-driven responses

### Logging
- Member joins
- Member leaves
- Message edits
- Message deletions
- Nickname changes
- Role updates
- Moderation actions

## Project Vision

Project Judy aims to create a Discord bot that feels less like a command interface and more like a genuine member of the community.

The long-term objective is to build a personality-driven assistant that naturally participates in servers while providing reliable moderation, automation, and AI-powered features.
