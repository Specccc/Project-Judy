# Project Judy

Project Judy is a modular Discord bot written in Python using discord.py.

## Features

- Modular Cog architecture
- Google Gemini AI integration
- SQLite database
- Slash commands
- Logging system
- Diagnostics
- Welcome system
- Future image generation support

## Project Structure

## Project Vision

Project Judy aims to create a Discord bot that feels less like a command interface and more like a genuine member of the community.

The long-term objective is to build a personality-driven assistant that naturally participates in servers while providing reliable moderation, automation, and AI-powered features.
